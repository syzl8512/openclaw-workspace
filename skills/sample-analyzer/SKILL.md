# 样本分析核心引擎

负责对样本文件进行标准化16维度分析，生成可复用的Skill。

## 功能

1. **梯度分析模式**
   - 快速模式（4维）：背景、目标、任务、量化数据
   - 深度模式（16维）：完整分析
   
2. **16维度分析框架**
   - 标准化分析维度
   - 提取关键信息
   - 生成结构化数据

3. **Skill生成**
   - 自动生成写作模板
   - 话术规范
   - 必备要素

## 触发方式

```
/样本分析 [文件路径]           # 快速模式
/样本分析 [文件路径] --deep    # 深度模式
```

## 分析维度

### 快速模式（4维）

| 维度 | 说明 |
|------|------|
| 1. 背景 | 政策/业务背景 |
| 2. 目标 | 核心目标 |
| 3. 任务 | 重点任务 |
| 4. 量化数据 | 关键指标 |

### 深度模式（16维）

| 维度 | 说明 |
|------|------|
| 1 | 政策背景/业务背景 |
| 2 | 核心目标 |
| 3 | 重点任务 |
| 4 | AI/数字化亮点 |
| 5 | 产业/业务影响 |
| 6 | 投资/预算机会 |
| 7 | 联通产品匹配 |
| 8 | 要点总结 |
| 9 | 量化指标 |
| 10 | 执行路径 |
| 11 | 风险评估 |
| 12 | 历史对比 |
| 13 | 区域/部门分析 |
| 14 | 时间节点 |
| 15 | 竞品/对标分析 |
| 16 | 可行性评估 |

## 错误处理

| 错误 | 处理 |
|------|------|
| 文件不存在 | 提示文件路径错误 |
| 解析失败 | 降级到纯文本处理 |
| 分析超时 | 保存部分结果，提示用户 |
| 生成Skill失败 | 仅保存分析结果 |
