# OpenClaw Personal Workspace

个人数字化生活管理系统

## 目录结构

```
├── skills/      # 技能库 - AI Agent 可调用的工具集
├── memory/      # 记忆库 - 长期记忆与知识沉淀
├── docs/        # 文档库 - 知识库与使用文档
├── Business/    # 职业轨道 - 工作相关模块
└── Personal/    # 生活轨道 - 个人生活模块
```

## 设计理念

- **模块化**：每个目录独立运作，可单独扩展
- **数据流动**：跨模块数据关联与洞察
- **AI驱动**：skills 提供能力，memory 实现学习
