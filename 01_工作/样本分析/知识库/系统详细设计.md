### 1.2 核心模块

| 模块 | 职责 | 关键技术 |
|------|------|----------|
| **文件接收器** | 接收飞书文件/本地文件 | 飞书SDK/文件监控 |

| **缓存管理器** | Hash校验避免重复解析 | 文件Hash |
| **归档器** | 自动化文件管理 | 文件操作 |

```
6. extract_metrics()        # 维度4
7. generate_skill(analysis_result)
8. vectorize(public)
9. archive(files)

深度模式（16维）：
1-4. 同快速模式
5. extract_ai_digital()     # 维度5

→ 后续流程同快速模式
```

```bash
# 5. 创建向量表
CREATE TABLE samples (
    id UUID PRIMARY KEY,
    title TEXT,
    content TEXT,
    embedding vector(1024),

CREATE INDEX ON samples USING ivfflat (embedding vector_cosine_ops);
```

## 七、缓存机制

### 7.1 Hash校验

```python
import hashlib

def calculate_content_hash(file_path: str) -> str:
    """计算文件内容Hash"""
    hasher = hashlib.sha256()
    with open(file_path, 'rb') as f:
        for chunk in iter(lambda: f.read(8192), b''):
            hasher.update(chunk)
    return hasher.hexdigest()

def check_cache(file_path: str) -> bool:
    """检查是否已有缓存"""
    content_hash = calculate_content_hash(file_path)
    # 查询Supabase中是否存在相同Hash的记录
    return supabase.query(
        "SELECT id FROM samples WHERE file_info.content_hash = ?",
        content_hash
    )
```
